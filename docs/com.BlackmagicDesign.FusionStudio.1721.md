## Description
<h2>Fusion Studio 17.2.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed a potential lag when using Fusion custom tools.</li>
<li>Addressed incorrect preview results with viewer LUTs.</li>
<li>Addressed an issue with loading LUT files in drfx bundles.</li>
<li>Addressed an issue with undo when entering values in color controls.</li>
<li>Addressed an issue with updating new content for templates.</li>
<li>Addressed an issue with OFX blocklist sometimes ignoring the last entry.</li>
<li>Addressed an issue with HotSpot occlusion masks and integer images.</li>
<li>Addressed an issue displaying groups in the macro editor.</li>
<li>Addressed an issue where importing ALUT3 files would not work.</li>
<li>Addressed incorrect loader behavior with some image sequences.</li>
<li>General performance and stability improvements.</li>
</ul>
