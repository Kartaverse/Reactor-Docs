## Description
<h2>Fusion Studio 17.4</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Text+ support for combined glyphs and right to left language layouts.</li>
<li>Improved vertical Text+ layout, vertical glyphs, rotation and line direction.</li>
<li>Improved Text+ character grouping, spacing, underlining and borders.</li>
<li>Text+ supports per-character stylistic sets for supported fonts.</li>
<li>Support for OpenType features, including old-style numbers in Text+.</li>
<li>Multiple Text+ improvements for character styling and animation.</li>
<li>Support for reading per-frame Blackmagic RAW metadata.</li>
<li>Improved default tracker search and pattern size for grid warps.</li>
<li>New search area scale slider in tracker options.</li>
<li>Improved memory usage when rendering long comps.</li>
<li>Improved support for empty loaders in macros.</li>
<li>Stability improvements to Anim Curves and XY Paths.</li>
<li>Addressed control value flickering with Keyframe Stretchers.</li>
<li>General performance and stability improvements.</li>
</ul>
