## Description
<h2>Fusion Studio 20.1.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed issues with loading and importing PSDs.</li>
<li>Addressed issue with single-frame sequences in Loader.</li>
<li>Addressed issue with Loader start times after changing global range.</li>
<li>Addressed issue with enabling aux channels in Swizzler.</li>
<li>Addressed Magic Mask sometimes deleting unrelated folders.</li>
<li>Addressed issue with saving versions in macros.</li>
<li>General performance and stability improvements.</li>
</ul>
