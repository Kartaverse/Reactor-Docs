## Description
<h2>Fusion Studio 17.4.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed nested macros not rendering or showing previews.</li>
<li>Addressed an issue displaying certain unicode characters in Text+.</li>
<li>Addressed incorrect viewer behavior with tablet input devices.</li>
<li>Addressed an issue where Text+ expressions would not work.</li>
<li>Addressed slow playback for Text+ with character styling.</li>
<li>Addressed a startup issue with display scalings of more than 250%.</li>
<li>Addressed an issue where video monitoring would not work correctly.</li>
<li>General performance and stability improvements.</li>
</ul>
