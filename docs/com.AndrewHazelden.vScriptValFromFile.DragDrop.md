## Description
<p>The "Vonk vScriptValFromFile DragDrop.fu" file allows you to import a ScriptVal Lua table .table file or a Concise Binary Object Representation (CBOR) file .cbor by dragging it into the Nodes view from a desktop Explorer/Finder/Linux folder browsing window. This is a quick way to bring external data records into your Resolve/Fusion composite. The DragDrop file supports dragging in multiple ScriptVal elements at the same time, and each item will be imported into a separate vScriptVal node.</p>

<p>This DragDrop file has a dependency on the Vonk FusionScriptVal atom package.</p>
