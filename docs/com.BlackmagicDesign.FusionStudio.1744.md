## Description
<h2>Fusion Studio 17.4.4</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Added support for Blackmagic RAW SDK 2.3.</li>
<li>Addressed an issue with 3D material memory usage.</li>
<li>Addressed an issue with some stereo viewer modes.</li>
<li>Addressed an issue with add tool dialog positioning.</li>
<li>Addressed an issue with tall aspect clips in monitor view.</li>
<li>Addressed an issue with Text+ outline perspective thickness.</li>
<li>Addressed some tools not updating to new timeline resolutions.</li>
<li>Addressed an issue when pressing alt/opt+K in planar tracker.</li>
<li>Addressed an issue writing frame rates in some QuickTime clips.</li>
<li>Addressed an issue with some duplicate color control labels.</li>
<li>General performance and stability improvements.</li>
</ul>
