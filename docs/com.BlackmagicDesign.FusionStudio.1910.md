## Description
<h2>Fusion Studio 19.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>New uExport tool to export USD scenes from compositions.</li>
<li>Polygon and B-spline viewer toolbar controls for multi poly tool.</li>
<li>New switch and uSwitch to manage and keyframe multiple inputs.</li>
<li>OCIO display transforms can now be applied with a tool.</li>
<li>OCIO CDL transforms can now be used as view LUTs.</li>
<li>Support for USD 24.08 with improved MaterialX performance.</li>
<li>Up to 3x faster performance with Fusion resize.</li>
<li>Improved performance with multiple and long paintbrush operations.</li>
<li>Support for freeze frame with TimeSpeed.</li>
<li>General performance and stability improvements.</li>
</ul>
