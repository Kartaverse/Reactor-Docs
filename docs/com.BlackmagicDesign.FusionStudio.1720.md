## Description
<h2>Fusion Studio 17.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Dramatically improved application startup performance.</li>
<li>Support for Fusion template bundles.</li>
<li>Support for UI layout presets for single and dual displays.</li>
<li>Improved support for offset modifier angle.</li>
<li>Addressed incorrect optical flow vector behavior in a static time region.</li>
<li>Addressed an issue with node name conflicts when importing certain SVG files.</li>
<li>Addressed an issue where deleting masks in groups would lose node connection.</li>
<li>Addressed an issue when using multiple viewer LUTs.</li>
<li>Addressed an issue with incorrect offset modifier angle calculation.</li>
<li>General performance and stability improvements.</li>
</ul>
