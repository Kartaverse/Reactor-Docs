## Description
<h2>Fusion Studio 8.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Fusion Studio for Linux</li>
<li>Fusion Render Node for Linux</li>
<li>UI and workflow improvements</li>
<li>General performance and stability improvements</li>
<li>Improved DNG support: Resolve matching</li>
<li>Double click in the empty area of comp tabs to create a new comp</li>
<li>New animation indicator shortcuts:</li>
<li>double-click to animate, or set key if already animated</li>
<li>ctrl/cmd+double-click to add expression</li>
<li>shift+double-click to remove modifiers/expressions</li>
<li>Can now add expressions to non-master colour wheel inputs. The expression edits will only be visible for the currently active wheel range (ie. master, low, mid, high).</li>
<li>Changed default font for Text+ and Text3D to Open Sans, for platform consistency</li>
<li>Can now specify multiple license servers (semicolon-separated) in FUSION_LICENSE_SERVER env var, * to scan local subnet (8.1)</li>
<li>Added Global Options/Background Color options to image view's menu</li>
<li>Added Save Version to the File menu</li>
<li>Can now set Gamut control custom values with a context menu</li>
<li>Can turn off image view description/RoI/DoD labels.</li>
<li>F1 brings up the PDF help files, with context awareness where possible</li>
<li>New Hotkey Manager</li>
</ul>
