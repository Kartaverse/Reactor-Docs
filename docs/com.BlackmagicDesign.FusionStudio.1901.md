## Description
<h2>Fusion Studio 19.0.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed issue with incorrect templates path map on Mac systems.</li>
<li>Addressed Text+ character level animation issue with keyframes view visible.</li>
<li>Addressed issue with full screen image viewers.</li>
<li>Addressed issue with loading large numbered files in a DRFX.</li>
<li>Addressed issue with ACES Transform viewlut on some tools.</li>
<li>Improved font matching for SVG imports.</li>
<li>General performance and stability improvements.</li>
</ul>
