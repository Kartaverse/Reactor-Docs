## Description
<center><h2>Despiller Plus</h2></center>
<p>An advanced despiller plugin for Resolve/Fusion v16 to v20+.</p>

Features:
<ul>
<li>Despill any color</li>
<li>Multiple spill controls (Bias, Strength, Threshold, Mix)</li>
<li>Restore the luminance from the color you despilled</li>
<li>Neutralize the despilled color to make it truly grayed</li>
<li>Tint the despilled color to your liking</li>
<li>Add an image background to color the despilled areas to make your footage fit right in the scene</li>
</ul>
