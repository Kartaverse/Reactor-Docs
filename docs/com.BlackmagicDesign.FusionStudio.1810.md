## Description
<h2>Fusion Studio 18.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Magic Mask.</li>
<li>Up to 10x faster Text+ playback and render performance.</li>
<li>Search for Fusion nodes using common keywords and categories.</li>
<li>Faster Fusion VariBlur with GPU acceleration.</li>
<li>Faster Fusion FilmGrain with GPU acceleration.</li>
<li>Support for ARRI LogC4 and ARRI Wide Gamut 4.</li>
<li>General performance and stability improvements.</li>
</ul>
