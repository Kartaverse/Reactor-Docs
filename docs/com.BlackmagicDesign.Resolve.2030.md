## Description
<h2>Resolve Free 20.3</h2>
<p>For your convenience, here are download links to Resolve Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Up to 32K resolution support with Apple M5 processors.</li>
<li>Improved performance for Resolve FX Noise Reduction.</li>
<li>Timeline backups now allow named snapshots for versioning.</li>
<li>Support for adding metadata fields as media pool bin columns.</li>
<li>Media Pool metadata ALE import and export support.</li>
<li>Option to create custom metadata for unknown fields on import.</li>
<li>Exported metadata files now include custom metadata fields.</li>
<li>Media pool views and state are now retained for each project.</li>
<li>Edit menu action to insert gap at playhead.</li>
<li>New assignable key shortcut in media pool to start a search.</li>
<li>Assignable key shortcut for open in timeline with source viewer.</li>
<li>Speed change and duration change actions now follow sync lock.</li>
<li>Added 2.39 and 2.40 broadcast safe aspect ratios.</li>
<li>Alpha support for film look creator, film damage and analog damage.</li>
<li>Support for IMF workflow for HDR Vivid and Audio Vivid.</li>
<li>Ability to embed HDR10+ metadata in QuickTime and MP4 encodes.</li>
<li>Ability to set stereoscopic mode in clip attributes for MXF clips.</li>
<li>Improved match frame behavior for clips with negative speeds.</li>
<li>Addressed missing clip actions in subtitle caption context menu.</li>
<li>Addressed intermittent lag when using stacked timelines.</li>
<li>Addressed source timeline not restored when switching back to edit.</li>
<li>Addressed issue tracking with high-quality deinterlace in Fusion.</li>
<li>Addressed issue with immersive world pose if using Fusion tools.</li>
<li>Addressed issue loading some deep OpenEXR files.</li>
<li>Addressed decode issues with compressed ARW clips.</li>
<li>Addressed AI Music Remixer muting audio on macOS 26.1.</li>
<li>Addressed possible QuickTime decode hang on Windows.</li>
<li>Addressed Remote Monitor start up issue on Windows.</li>
<li>General performance and stability improvements.</li>
</ul>
