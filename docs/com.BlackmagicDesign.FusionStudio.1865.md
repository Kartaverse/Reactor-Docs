## Description
<h2>Fusion Studio 18.6.5</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed a mesh point update issue when changing grid warp size.</li>
<li>Addressed a crash using an invalid MaterialX USD material.</li>
<li>Addressed issue updating USD shape transforms for selected prims.</li>
<li>Addressed issue with HDR option visibility in color space transform tool.</li>
<li>Addressed issue with removing expression from inspector plus button.</li>
<li>General performance and stability improvements.</li>
</ul>
