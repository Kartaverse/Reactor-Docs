## Description
<h2>Fusion Studio 20.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for new immersive option in the viewer 360 Views menu.</li>
<li>Support for new immersive patcher tool.</li>
<li>Support for Magic Mask v2 in Fusion Studio.</li>
<li>Edit individual tool controls from the inspector context menu.</li>
<li>Support for caching deep composites to disk.</li>
<li>Improved DoD and RoI support for deep image compositing tools.</li>
<li>Improvements for USD Renderer for overscan.</li>
<li>Support for MultiText indent, align and paragraph level styling.</li>
<li>Support for built in MultiText character level styling.</li>
<li>Ability to import CSV files as column-aligned MultiText text boxes.</li>
<li>Improved sDuplicate with style, jitter and aligned rotations.</li>
<li>Support for swizzler to create alpha only multi layer outputs.</li>
<li>Support for swizzler background channels.</li>
<li>Ability to isolate layers in LayerRegex using named captures.</li>
<li>Support for regular expression based selection in Cryptomatte.</li>
<li>Support for Blackmagic RAW SDK 5.0.</li>
<li>General performance and stability improvements.</li>
</ul>
