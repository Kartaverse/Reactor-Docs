## Description
<h2>Fusion Studio 19.1.4</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic RAW SDK 4.5.</li>
<li>Addressed issue with loading Text+ with looped CLS animation.</li>
<li>Addressed issue with sMerge changing layer order.</li>
<li>Addressed issue with saving comps after importing SVG files.</li>
<li>Addressed issue with some GPU preference device settings.</li>
<li>General performance and stability improvements.</li>
</ul>
