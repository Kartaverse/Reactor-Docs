## Description
<h2>Fusion Studio 18.6.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Improved speed of transforming objects with MaterialX materials.</li>
<li>Addressed issue with uReplaceMaterial improperly applying textures.</li>
<li>Addressed issue when renaming uMerge.</li>
<li>Addressed issue with opacity when changing USD object materials.</li>
<li>Addressed issue with some text+ image mapping cases.</li>
<li>Addressed issue with text+ follower caching.</li>
<li>Addressed issue with animating paint brush size.</li>
<li>Addressed an issue with paint cloning artefacts.</li>
<li>Addressed issues with clipping and transforms in OCIO viewer LUTs.</li>
<li>Addressed issue with using multiple view LUTs.</li>
<li>Addressed issue with flickering in some transform cases.</li>
<li>Addressed issue with scroll wheel modifiers in spline and keyframes views.</li>
<li>Addressed an issue with 2D particle subframe accuracy.</li>
<li>Addressed an issue with particle bitmap bounce regions.</li>
<li>Addressed issues with tracker's animated preview thumbnail.</li>
<li>Addressed issue with reverse tracking previews.</li>
<li>Addressed issues with magic mask bidirectional tracking.</li>
<li>Addressed an issue with strokes canceling magic mask tracking.</li>
<li>Addressed issue with creating roto projects in BorisFX Silhouette.</li>
<li>Addressed an issue with Open FX plugin icons.</li>
<li>General performance and stability improvements.</li>
</ul>
