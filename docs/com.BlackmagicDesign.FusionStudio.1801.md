## Description
<h2>Fusion Studio 18.0.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed a color issue with Mandelbrot on Apple silicon.</li>
<li>Addressed a language translation issue with user controls.</li>
<li>Addressed an issue with gain and gamma when normalizing viewers.</li>
<li>Addressed a blend and jittering issue on duplicate with Apple silicon.</li>
<li>Addressed a gain issue with duplicate in some cases.</li>
<li>General performance and stability improvements.</li>
</ul>
