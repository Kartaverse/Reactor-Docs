## Description
<h2>Fusion Studio 20.2.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed issue editing spline ease in values.</li>
<li>Addressed some duplicate viewer menu entries.</li>
<li>Addressed issue when loading some USD texture paths.</li>
<li>Addressed viewing named layers from Renderer3D.</li>
<li>Addressed aperture metadata naming in OpenEXR files.</li>
<li>Addressed net rendering of some Swizzler comps.</li>
<li>Addressed issue when decoding jpeg gain maps.</li>
<li>Addressed issue viewing short images on Blackmagic monitoring devices.</li>
<li>General performance and stability improvements.</li>
</ul>
