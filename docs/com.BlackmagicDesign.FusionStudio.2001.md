## Description
<h2>Fusion Studio 20.0.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed DeckLink preview issues.</li>
<li>Addressed issues with loading PSD layers.</li>
<li>Addressed issues with saving nodes with custom layer lists.</li>
<li>Addressed missing Fuse examples in the Developer folder.</li>
<li>Addressed multiple OpenColorIO issues.</li>
<li>Addressed issue with Object Removal tool when switching comps.</li>
<li>Addressed incorrect output when adjusting Smooth UV with Vector Warp.</li>
<li>General performance and stability improvements.</li>
</ul>
