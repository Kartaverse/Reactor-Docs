## Description
<h2>Fusion Studio 18.1.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue when trimming some comps with anim curves.</li>
<li>Addressed an issue loading paths with no locked keyframes.</li>
<li>Addressed an issue with middle mouse scrolling the macro editor.</li>
<li>Addressed an issue when trimming comps with background media.</li>
<li>Addressed an issue with LuaModules and path maps in the render node.</li>
<li>General performance and stability improvements.</li>
</ul>
