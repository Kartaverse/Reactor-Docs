## Description
<h2>Fusion Studio 16.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic Pocket Cinema Camera presets in the Camera3D and CameraTracker tools</li>
<li>Support for ACEScg as a gamut option</li>
<li>Support for using Shift + drag to set path keyframes when using non-spline displacement modifiers</li>
<li>Improved memory management on multi-GPU systems</li>
<li>Improved viewer interactivity and playback performance in single GPU systems</li>
<li>Improved render all savers status display</li>
<li>Addressed an issue where the viewer would not show real time changes in the Color Corrector node when separate color wheels were used</li>
<li>Addressed an issue where applying multiple OCIO view LUTs would show erroneous behavior</li>
<li>Addressed an issue where auto arranging tools in the node editor would sometimes cause an overlap of nodes</li>
<li>Addressed an issue where saving Fusion preferences when viewing the 3D viewer would disable the lighting options in the 3D viewer</li>
<li>Addressed an issue where the Stereo context menu would not accurately reflect the checkerboard and interlaced viewer modes</li>
<li>Addressed an issue where inaccessible paths in Windows %PATH% environment variable would cause a crash on startup</li>
<li>Addressed an issue where adding images with different resolutions to some OpenFX plugins would cause a crash</li>
<li>Addressed an issue where color dodge and burn would produce inverted results if the input image had invalid values</li>
<li>Addressed an issue where the White Balance tool would sometimes show incorrect results when using GPU processing</li>
<li>General performance and stability improvements</li>
</ul>
