## Description
<h2>Fusion Studio 17.4.5</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue with color corrector's ranges control.</li>
<li>Addressed an issue with missing Text+ fonts in Monterey.</li>
<li>Addressed issues when extruding 3D text from specific fonts.</li>
<li>Improved performance of the bitmap mask tool.</li>
<li>Addressed random seed value changes when switching duplicate versions.</li>
<li>Addressed expressions on letterbox's Pixel Aspect when loading comps.</li>
<li>Addressed an issue with AllowDuplicate in the LoadComp API.</li>
<li>General performance and stability improvements.</li>
</ul>
