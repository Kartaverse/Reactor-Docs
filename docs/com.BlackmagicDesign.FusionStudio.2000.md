## Description
<h2>Fusion Studio 20.0</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>New Deep image compositing toolset.</li>
<li>Multi layer pipelining for OpenEXR, PSD and stereoscopic 3D.</li>
<li>Vector warping toolset for image patching and cleanup.</li>
<li>MultiText tool with layers and easy transform, clip and wrap controls.</li>
<li>Text+ paragraph, line wrapping and bounding tools.</li>
<li>Support for 180 VR.</li>
<li>Support for ACES 2.0.</li>
<li>Support for OCIO 2.4.2 with ACES 2.0 support.</li>
<li>3D Scene Dome Light.</li>
<li>Support for native Cryptomatte workflows.</li>
<li>Support for Film Look Creator.</li>
<li>Faster GPU based pano map and spherical stabilizer tools.</li>
<li>Ability to search tool based on effects category.</li>
<li>General performance and stability improvements.</li>
</ul>
