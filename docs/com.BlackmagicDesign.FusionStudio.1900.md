## Description
<h2>Fusion Studio 19.0</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Resolve FX Surface Tracker.</li>
<li>Support for Resolve FX Object Removal.</li>
<li>Multi Poly tool for easier rotoscoping of complex objects.</li>
<li>IntelliTrack AI point tracker option in Tracker.</li>
<li>Support for Open Color IO 2.3.</li>
<li>USD enhancements for Material X and advanced volumetric shading.</li>
<li>Multiple shape toolset enhancements.</li>
<li>Improved Text+ inspector and on-screen controls.</li>
<li>Show user defined guides in the viewer.</li>
<li>Option to specify duplication path in duplicate tools.</li>
<li>Support for deleting and resetting layers in MultiMerge.</li>
<li>Option for sBooleans to retain the upstream style.</li>
<li>Set control to default now clears animations, modifiers and expressions.</li>
<li>Track forward and reverse controls for multiple tracker tools.</li>
<li>Ability to match planar tracker sizes to dissimilar logo and image overlays.</li>
<li>New viewLUT for ACES Transform.</li>
<li>Faster rendering of streaming 3D textures in compositions.</li>
<li>Up to 3x faster Magic Mask performance.</li>
<li>Up to 2.5x faster DaVinci Neural Engine performance with Nvidia TensorRT.</li>
<li>Improved performance for some large Fusion compositions.</li>
<li>New left flow and mid flow UI presets with vertical viewer layouts.</li>
<li>Option to edit right digit when editing inspector values using cursor keys.</li>
<li>General performance and stability improvements.</li>
</ul>
