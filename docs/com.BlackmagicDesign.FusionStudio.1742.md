## Description
<h2>Fusion Studio 17.4.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue with focus on Fusion viewers with Wacom pen taps.</li>
<li>Addressed invisible slider issue when single tapping using Wacom tablet.</li>
<li>Addressed issue with error when loading a project with fuses.</li>
<li>Addressed startup crash on systems with custom location installations.</li>
<li>Addressed issue with changing ObjectID for FBX objects with override 3D.</li>
<li>Addressed an issue with image mapping on character styled text.</li>
<li>Addressed an issue when deleting keyframes in the splines view.</li>
<li>Addressed an issue with importing large FBX files.</li>
<li>Addressed an issue with loading glyph advances for certain Type 1 fonts.</li>
<li>Addressed issues with frame scripts.</li>
<li>Addressed an issue when creating a preview.</li>
<li>Addressed an issue with fullscreen flipbooks.</li>
<li>General performance and stability improvements.</li>
</ul>
