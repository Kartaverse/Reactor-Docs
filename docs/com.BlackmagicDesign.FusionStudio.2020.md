## Description
<h2>Fusion Studio 20.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Improved MultiText align and transformation controls.</li>
<li>Support for immersive option in PanoMap.</li>
<li>Support for animated image inputs for USD tools.</li>
<li>Multilayer support for 3D renderer.</li>
<li>Multilayer AOV channel support for USD renderer.</li>
<li>Support for lens space EXR metadata.</li>
<li>Up to 2x faster surface tracker performance.</li>
<li>Addressed magic mask cache issues with negative frame offsets.</li>
<li>General performance and stability improvements.</li>
</ul>
