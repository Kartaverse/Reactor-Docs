## Description
<h2>Fusion Studio 18.1.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic RAW SDK 2.8.</li>
<li>Addressed a Text3D issue with characters with intersections.</li>
<li>Addressed an issue importing tracking data in Fusion.</li>
<li>Addressed an issue evaluating Group or Macro tool expressions.</li>
<li>Addressed an issue with Text+ on old CPUs.</li>
<li>General performance and stability improvements.</li>
</ul>
