## Description
<h2>Fusion Studio 19.1.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed issue with saving tools with some edited controls.</li>
<li>Addressed issue with outer poly rendering on linear b-splines.</li>
<li>Addressed issue with loading from disk caches on macros.</li>
<li>Addressed issue with using disk caches on transformation tools.</li>
<li>Addressed issue with hover preview when an effect is used.</li>
<li>Addressed issue with saving some OpenEXR files after disk caching.</li>
<li>Addressed issue with rendering a SphereMap in software.</li>
<li>Addressed issue with reducing points on Hue Curves.</li>
<li>Addressed issues with time controls in Calculation and Offset.</li>
<li>General performance and stability improvements.</li>
</ul>
