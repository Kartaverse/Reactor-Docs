## Description
<h2>Fusion Studio 20.2.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed MultiText entry issue for Chinese, Japanese or Korean.</li>
<li>Addressed invalid values with the Cook-Torrance 3D material.</li>
<li>Addressed crash in Cryptomatte while picking.</li>
<li>Addressed issue with animations upstream of uReplaceMaterial.</li>
<li>Addressed Film Look Creator animation with static sources.</li>
<li>Addressed Immersive Patcher artefacts for large angle of view.</li>
<li>General performance and stability improvements.</li>
</ul>
