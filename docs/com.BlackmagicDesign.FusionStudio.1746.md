## Description
<h2>Fusion Studio 17.4.6</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Ability to execute startup script commands from terminal.</li>
<li>Support for CopySettings to work on modifiers.</li>
<li>Improved stability with large 3D renders on M1 Max.</li>
<li>Addressed SVG groups going offline if duplicate node names exist.</li>
<li>Addressed an issue with malformed fonts.</li>
<li>Addressed an issue extruding 3D text from specific fonts.</li>
<li>General performance and stability improvements.</li>
</ul>
