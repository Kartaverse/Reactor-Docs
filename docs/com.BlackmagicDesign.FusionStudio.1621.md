## Description
<h2>Fusion Studio 16.2.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue where OCIO tools would not process canvas colour.</li>
<li>Addressed an issue where the paint color picker would not work.</li>
<li>Addressed an issue when DeltaKeyer source with solid matte exceeded RoI.</li>
<li>Addressed an issue where some fonts would not be available for text tools on Mac.</li>
<li>Addressed an issue where extracting mask fields in a comp could cause a crash.</li>
<li>Addressed an issue where ResolveParameter was unavailable in Fusion Studio.</li>
<li>Addressed an issue with executing some plugins and scripts on Mac Mojave.</li>
<li>Addressed an issue where CustomData would sometimes be stored incorrectly.</li>
<li>General performance and stability improvements.</li>
</ul>
