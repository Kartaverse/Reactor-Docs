## Description
<h2>Fusion Studio 17.1.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue when loading splines set to ping-pong duplicate.</li>
<li>Addressed incorrect brightness for ambient light in 3D Camera setups.</li>
<li>Addressed an issue with directional light shadows in some projects.</li>
<li>Addressed an issue with spin and rotation assignments to new particles.</li>
<li>General performance and stability improvements.</li>
</ul>
