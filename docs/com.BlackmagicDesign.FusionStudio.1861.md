## Description
<h2>Fusion Studio 18.6.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed issue with certain fonts in text 3D.</li>
<li>Addressed issue with grid depth occlusion in the USD viewer.</li>
<li>Addressed issue with MaterialX normal map artefacts in USD.</li>
<li>Addressed issue with moving locked point selections in the splines view.</li>
<li>Addressed issue with disappearing key markers in the splines view.</li>
<li>Addressed issue with animated grid warp caching.</li>
<li>Addressed issue with combining and union of various shapes.</li>
<li>General performance and stability improvements.</li>
</ul>
