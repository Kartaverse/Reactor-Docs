## Description
<h2>Fusion Studio 16.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for write on/off, end-cap style and miter limit in mask tools.</li>
<li>Support for a "show folder" option in the Fusion effects Library.</li>
<li>Support for showing macros in the Fusion effects library.</li>
<li>Support for inverting the temporal behavior of the keyframe stretcher tool.</li>
<li>Support for resolve parameter modifier, for transition template animation.</li>
<li>Support for using DaVinci Resolve dongles with Fusion Render Nodes.</li>
<li>Support for RED SDK 7.2.1 and decoding clips from the Komodo camera.</li>
<li>Improved OpenEXR support on Linux.</li>
<li>Improved persistence of view settings.</li>
<li>Improved kerning of some Adobe type 1 fonts in text+ and text 3D tools.</li>
<li>Improved display of localized font names in text+ and text 3D tools.</li>
<li>Improved performance with gradient background now in GPU.</li>
<li>Improved performance with dve now in GPU.</li>
<li>Improved performance with corner pin now in GPU.</li>
<li>Improved performance with stereo combining in 3D renderer now in GPU.</li>
<li>General performance and stability improvements.</li>
</ul>
