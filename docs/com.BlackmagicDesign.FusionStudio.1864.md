## Description
<h2>Fusion Studio 18.6.4</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic RAW SDK 3.6.</li>
<li>Addressed issue with 2D particle positions in image emitters.</li>
<li>Addressed USD loader trim and time scale issue for new assets.</li>
<li>Addressed issue with updating overlay control positions.</li>
<li>Addressed issue with locator 3D marker rendering.</li>
<li>Addressed a possible crash when using certain expressions.</li>
<li>Addressed issue with viewer scale affecting set region.</li>
<li>Addressed issue with clipping of lens aberrations in hot spot.</li>
<li>Addressed issue with auto-detection of GPU memory.</li>
<li>Addressed issue with DoD image clipping with autoproxy enabled.</li>
<li>Addressed issue with French translation of Paste Settings.</li>
<li>Addressed issue with not fully disabling some controls.</li>
<li>Addressed issue with caching during playback.</li>
<li>General performance and stability improvements.</li>
</ul>
