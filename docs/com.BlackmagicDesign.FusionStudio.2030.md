## Description
<h2>Fusion Studio 20.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Improved 3D rendering of shadow, reflect, and refract layers.</li>
<li>Addressed issue with loading some Open FX plugins twice.</li>
<li>Addressed issue with some ping-pong splines.</li>
<li>Addressed issue closing Fusion with Python 3.14 installed.</li>
<li>Addressed issue with Magic Mask 2 tracks leaking memory.</li>
<li>Addressed issue with viewer overlays sometimes not updating.</li>
<li>Addressed issue with Planar Transform when proxy scaling.</li>
<li>Addressed issue with tool names when importing FBX twice.</li>
<li>General performance and stability improvements.</li>
</ul>
