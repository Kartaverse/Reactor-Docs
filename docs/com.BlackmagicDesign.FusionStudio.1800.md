## Description
<h2>Fusion Studio 18.0</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Fractional UI scaling support, including 150%, in preferences.</li>
<li>Support for setting user interface to French, Spanish and other languages.</li>
<li>Multi-button mode selection in inspector for multiple tools.</li>
<li>Support for all modern and future python 3 versions for scripting.</li>
<li>Support for OpenColorIO 2.1.1.</li>
<li>Support for OpenEXR 3.1.4.</li>
<li>Support for live previews when using the Text+ color picker.</li>
<li>New expression animated Custom Poly modifier for masks and strokes.</li>
<li>Faster GPU accelerated paint tool with smoother strokes.</li>
<li>Faster duplicate tool with additional blur, glow and size controls.</li>
<li>Improved Linux installation and compatibility.</li>
<li>Improved Open FX compatibility.</li>
<li>Improved text renders with better tessellation and cleaner outlines.</li>
<li>Multiple new composition blend modes.</li>
<li>Improved tracker pattern grabbing behavior.</li>
<li>Clearing Fusion disk cache is now more selective.</li>
<li>Improved OpenEXR zip compression type compatibility.</li>
<li>Support for H.265 renders from savers on Mac.</li>
<li>New dropdown menu for Macro editor window.</li>
<li>General performance and stability improvements.</li>
</ul>
