## Description
<h2>Fusion Studio 18.5</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for importing Universal Scene Descriptor (USD) files</li>
<li>Introduction of the USD toolset for managing USD assets.</li>
<li>Support for USD Hydra based renderers including Storm.</li>
<li>New multi-merge tool for compositing multiple layers.</li>
<li>Support for a native AI-based depth map tool.</li>
<li>Support for Blackmagic RAW SDK 3.3.</li>
<li>GPU accelerated Clean Plate.</li>
<li>GPU accelerated Anaglyph.</li>
<li>Up to 3x faster renders when using splitter tool.</li>
<li>Support for OpenEXR DWA compression level in saver nodes.</li>
<li>Support for BMD Film Gen 5 in CineonLog tool.</li>
<li>Support for previewing individual shape tools.</li>
<li>General performance and stability improvements.</li>
</ul>
