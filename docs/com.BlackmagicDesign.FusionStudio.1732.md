## Description
<h2>Fusion Studio 17.3.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic RAW SDK 2.2.</li>
<li>Addressed potential freezes after using tracker modifiers.</li>
<li>Addressed a planar transform issue on small images.</li>
<li>Addressed an issue with verbose logging.</li>
<li>Addressed a failure to save Fusion settings on new installs.</li>
<li>General performance and stability improvements.</li>
</ul>
