## Description
<h2>Fusion Studio 18.6</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Resolve FX Color transform, ACES transform.</li>
<li>Support for Resolve FX gamut mapping and gamut limiter.</li>
<li>Support for Resolve FX Relight.</li>
<li>Support for Resolve FX Chromatic Aberration Removal.</li>
<li>Support for Resolve FX Chromatic Adaptation.</li>
<li>Support for extruding and beveling 3D shapes.</li>
<li>Support for polygon shape tool.</li>
<li>Support for rendering scenes with MaterialX materials and OpenVDB volumes.</li>
<li>New MaterialX tools to load .mtlx files and apply them to objects.</li>
<li>Support for selecting and adjusting imported cameras and lights.</li>
<li>Support for selectively transforming and hiding USD objects.</li>
<li>New tool to select between variants of an USD object.</li>
<li>Improved USD Loader with trim, loop and reverse support.</li>
<li>New scene tree dialog for object selection in USD.</li>
<li>Context menu option in the nodes and keyframes views to reset tool trims.</li>
<li>Support for DaVinci Intermediate conversion options in Fusion tools.</li>
<li>General performance and stability improvements.</li>
</ul>
