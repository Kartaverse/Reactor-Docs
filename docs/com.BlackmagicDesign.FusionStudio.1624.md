## Description
<h2>Fusion Studio 16.2.4</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue with aux channels and combiner nodes.</li>
<li>Addressed an issue with slower render times for custom tools.</li>
<li>Addressed an issue with gradient modifiers applied on color controls.</li>
<li>Addressed incorrect results when using gain controls in the Duplicate fuse.</li>
<li>Addressed an issue with importing Blackmagic Design camera LUTs.</li>
<li>Multiple performance and stability improvements.</li>
</ul>
