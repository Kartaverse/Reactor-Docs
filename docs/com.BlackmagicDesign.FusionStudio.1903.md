## Description
<h2>Fusion Studio 19.0.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic RAW SDK 4.3.</li>
<li>Addressed issue with applying OCIO Looks.</li>
<li>Addressed issue with slow cascaded expressions in the Inspector.</li>
<li>Addressed issue with loading USD on some systems.</li>
<li>Addressed issue with pasting keyframes into empty Splines view.</li>
<li>Addressed issue with tabbing between fields in Preferences.</li>
<li>Addressed issue with slow caching of branched masks.</li>
<li>Addressed issue with DoD for non-closed inner/outer polygons.</li>
<li>Addressed issue with closing comps while Sticky Note is open.</li>
<li>General performance and stability improvements.</li>
</ul>
