## Description
<h2>Fusion Studio 19.0.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Added option for simpler color swatches in edited controls.</li>
<li>Addressed issue with inconsistent points when using sJitter and Extrude3D.</li>
<li>Addressed issue with uninstaller on Intel Macs.</li>
<li>Addressed issue with software rendering of 3D cube maps.</li>
<li>Addressed issue with empty inverted masks on merge tools.</li>
<li>Addressed issue with picking color when using CST or ACES view LUTs.</li>
<li>Addressed issue with using OpenColorIO for buffer LUTs.</li>
<li>Addressed issue with OCIOColorspace transform direction.</li>
<li>Addressed issue updating OCIOColorspace look transform list.</li>
<li>Addressed issue with triggering tracking via scripting.</li>
<li>Addressed issue with context help for some tools.</li>
<li>Addressed delta keyer spill suppression affecting wrong channels.</li>
<li>Addressed issue with outer polygon overlay visibility.</li>
<li>Addressed issue modifying a polygon after changing time with shape box.</li>
<li>Sticky note changes now mark the composition as modified.</li>
<li>General performance and stability improvements.</li>
</ul>
