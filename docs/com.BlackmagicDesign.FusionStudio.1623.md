## Description
<h2>Fusion Studio 16.2.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed incorrect masking in Text+ when using image shading.</li>
<li>Addressed incorrect behavior deleting gradient points in inspector.</li>
<li>Addressed issues with LUTs and bit formats causing blank previews.</li>
<li>Addressed issues with applying specific comp view preferences correctly.</li>
<li>Addressed some inconsistent saver file extensions behavior.</li>
<li>Multiple performance and stability improvements.</li>
</ul>
