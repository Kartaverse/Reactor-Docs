## Description
<h2>Fusion Studio 18.1.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Ability to pick color in glow, soft glow, and highlight tools.</li>
<li>Improved performance for Text+ set to zero opacity.</li>
<li>Addressed an issue handling bad utf-8 text in Text+.</li>
<li>Addressed an issue with encoding 12 bit DNxHR files.</li>
<li>Addressed an issue with paint smear strokes.</li>
<li>Addressed a formatting issue with missing tool warnings.</li>
<li>Addressed an issue with anaglyph alpha.</li>
<li>Addressed an issue loading some DNG files.</li>
<li>Addressed an issue with trimming comps with anim curves.</li>
<li>Addressed an issue with some depth maps in the shadow tool.</li>
<li>Addressed an inverted merge mask issue on recent AMD drivers.</li>
<li>General performance and stability improvements.</li>
</ul>
