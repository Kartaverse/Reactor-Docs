## Description
<h2>Fusion Studio 17.4.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed potential freezes with fuses on Apple Silicon.</li>
<li>Addressed a stability issue with classic optical flow on Apple Silicon.</li>
<li>Addressed an issue when deleting looped spline points.</li>
<li>Addressed an issue with pasting connected tools into a group.</li>
<li>Addressed an issue with 3D text sometimes not rendering.</li>
<li>Addressed a DVE rendering issue with some rotations with OpenCL.</li>
<li>Addressed a node editor redraw issue for instanced tools.</li>
<li>Addressed an issue with unreadable font files.</li>
<li>Addressed an issue with using styling on unstyled text fields.</li>
<li>General performance and stability improvements.</li>
</ul>
