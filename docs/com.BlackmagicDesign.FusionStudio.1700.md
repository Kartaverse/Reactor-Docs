## Description
<h2>Fusion Studio 17.0</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Anim curves modifier for convenient easing and shaping of animations.</li>
<li>Support for user customization of the Fusion toolbar above the node editor.</li>
<li>Initial support for a GPU accelerated 2D shapes toolkit for motion graphics.</li>
<li>Improved optical flow behavior in Fusion tools.</li>
<li>Support for bookmarks to switch between different views in the node editor.</li>
<li>Support for activation keys for Fusion Studio.</li>
<li>Support for improved inspector with tabs and keyframes.</li>
<li>Multiple performance and stability improvements.</li>
</ul>
