## Description
<h2>Fusion Studio 18.0.2</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Addressed an issue with scaling of the hue curves control.</li>
<li>Addressed an issue with bad edges on double polyline masks.</li>
<li>Addressed a repetitions issue with Mandelbrot on Apple silicon.</li>
<li>Addressed an issue with importing FBX into a default comp.</li>
<li>Addressed an issue with changing tracker modifier on transforms.</li>
<li>Addressed an issue with vector motion blur artefacts.</li>
<li>General performance and stability improvements.</li>
</ul>
