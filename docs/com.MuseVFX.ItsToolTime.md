# It's Tool Time!
___

## Category
Scripts/Comp

## Author
Bryan Ray

## Version
1.0

___

## Description
<p>This script gets a list of all tools in the comp and how long their most recent renders took, and it displays that information in a GUI.</p>

by Bryan Ray for Muse VFX
