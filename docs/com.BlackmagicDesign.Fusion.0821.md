## Description
<h2>Fusion Free 8.2.1</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>UI and workflow improvements</li>
<li>General performance and stability improvements</li>
<li>Fixed Render Node crash on exit</li>
<li>Edit box contents are reselected after pressing enter if Auto Control Advance is disabled</li>
<li>Operations in range control edit boxes weren't being evaluated after pressing enter</li>
<li>Fixed creation of undo events for multiline text edits</li>
<li>Ensure bin state is saved on exit</li>
<li>Stop an occasional inadvertent attempt to replace a tool with itself when pasting in the flow view</li>
<li>Macro tools weren't available for toolbar customisation until after the Tools menu had been used</li>
<li>Comp file requester would always browse to Comps: rather than the path of the last loaded comp</li>
<li>Fixed flow group background in HiDPI/Retina</li>
<li>OFX plug requests for images outside of normal render processing will now get them at the current comp quality instead of always high-quality, non-proxy</li>
<li>Allow OFX plugins to force interactive rendering in some cases</li>
</ul>
