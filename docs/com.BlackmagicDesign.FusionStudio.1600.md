## Description
<h2>Fusion Studio 16.0</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Fusion Studio 16 release concurrent with DaVinci Resolve 16</li>
<li>Support for Metal and CUDA GPU processing modes</li>
<li>Improved playback performance</li>
<li>Improved 3D engine performance</li>
<li>Support for the latest Blackmagic RAW SDK 1.4</li>
<li>Support for MultiSampling in the 3D Renderer</li>
<li>Ability to flexibly scale animation with changing clip lengths</li>
<li>Improved responsiveness of Fusion viewer interactions when using a Wacom tablet</li>
<li>Improved performance for Change Depth with new Error Diffusion option</li>
<li>Improved performance and accuracy for Camera Tracker, with support for lens distortion</li>
<li>Improved performance for Dissolve with GPU acceleration support</li>
<li>Improved performance for Duplicate tool with GPU acceleration support</li>
<li>Improved performance for TimeSpeed and TimeStretcher with GPU acceleration support</li>
<li>Improved performance for Planar Tracker</li>
<li>Improved performance for B-Spline rendering with GPU acceleration support</li>
<li>Improved performance for Vector Motion Blur and motion blur in other tools with GPU acceleration support</li>
<li>Improved performance for Splitter and Combiner with GPU acceleration support</li>
<li>Improved performance for BitmapMask with GPU acceleration support</li>
<li>Improved performance for CornerPin and PerspectivePin with GPU acceleration support</li>
<li>Improved performance for Mask rendering with GPU acceleration support</li>
<li>Improved performance for Color operations with GPU acceleration support</li>
<li>Improved performance when selecting a paint tool with multiple strokes</li>
<li>Improved zoom behavior in the keyframe and spline panels</li>
<li>Improved performance and stability when rendering mask shapes when using the Metal GPU processing mode</li>
<li>Improved unicode PSD layer name support</li>
<li>Support for using DaVinci Resolve Studio dongles to run Fusion Studio</li>
<li>General stability and performance improvements</li>
</ul>
