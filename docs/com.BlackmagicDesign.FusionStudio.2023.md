## Description
<h2>Fusion Studio 20.2.3</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Support for Blackmagic Raw SDK 5.1.</li>
<li>Up to 4x faster Magic Mask and Depth Map on Intel GPUs./li>
<li>Addressed issue with deep image proxy scaling.</li>
<li>Addressed issue with deep image flipbook previews.</li>
<li>Addressed issue with saving some interlaced file formats.</li>
<li>General performance and stability improvements.</li>
</ul>
