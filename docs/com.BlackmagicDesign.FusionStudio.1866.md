## Description
<h2>Fusion Studio 18.6.6</h2>
<p>For your convenience, here are download links to Fusion Standalone full version releases that are available on the Blackmagic Design support page.</p>
<h2>What's new</h2>
<p>The following features have been added or updated.</p>
<ul>
<li>Improved render performance while viewing some tool configurations.</li>
<li>Improved performance loading some OpenEXR files.</li>
<li>Addressed issue with deselecting inspector fields.</li>
<li>Addressed issue exiting with Python 3.9.8 and higher.</li>
<li>Addressed issue with auto crop on empty tools.</li>
<li>Addressed issue with merging spawned particles.</li>
<li>Addressed issue with the add tool and LUT editor dialogs.</li>
<li>Addressed issue with path displacements on loading projects.</li>
<li>Addressed issue with FBX import skipping some keyframe.</li>
<li>General performance and stability improvements.</li>
</ul>
